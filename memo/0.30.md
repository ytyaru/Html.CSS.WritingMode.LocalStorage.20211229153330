# 0.30 ページ遷移を実装した（操作方法（キーボード、マウス、タッチ）。遷移アルゴリズム（前。次。最初。最後。0からはじまる絶対値指定）。計算（全ページ数。現在ページ位置）。スクロールバー非表示）

　本のようにページ遷移できるようにした。

* スクロールバーを非表示にした
* 1画面サイズ分をスクロールする
* ワンタッチでスクロールする

スクロールバー非表示用CSS
```css
    overflow-x:hidden;
    overflow-y:hidden;
```

　ページ遷移は次の状態でも可能になるよう配慮した。

* キーボードのみ
* マウスのみ
* タッチのみ

操作|キーボード|マウス|タッチ
----|----------|------|------
前|`p`|画面端クリック|画面端タップ
次|`n`|画面端クリック|画面端タップ
最初|`f`|-|-
最後|`l`|-|-
指定ページ位置|`m`|-|-

　キーボードは他にもキーアサインされている。詳細はpaging.jsのコードを参照。
